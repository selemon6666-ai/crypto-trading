# 🚀 Crypto Trading Simulator

A real-time **paper trading platform** where you can practice trading cryptocurrencies with $10,000 virtual money. No real money involved - perfect for learning!

## ✨ Features

- 💰 **$10,000 virtual balance** to start trading
- 📊 **Real-time price updates** every 3 seconds
- 🪙 **5 Cryptocurrencies**: Bitcoin (BTC), Ethereum (ETH), Binance Coin (BNB), Solana (SOL), Dogecoin (DOGE)
- 📈 **Buy/Sell orders** with instant execution
- 💼 **Portfolio tracker** showing your holdings
- 📉 **Net worth calculator** (balance + portfolio value)
- 🎨 **Beautiful, responsive design** works on mobile & desktop

## 🎮 How to Use

1. Select a cryptocurrency from the dropdown or click on any market price
2. Enter the amount you want to buy/sell
3. Click **BUY** or **SELL**
4. Watch your portfolio and net worth update instantly!
5. Prices update automatically every 3 seconds - try buying low and selling high!
